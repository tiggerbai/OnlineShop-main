# OnlineShop 前台

這是 OnlineShop 商業網站的前台部分，使用 .NET 5.0 開發。

## 功能

- 商品展示：顯示後台上架的商品，商品會依據分類顯示於商場的輪播區塊。

## 安裝

1. - .NET Core 3.1 或以上版本
   - SQL Server
2. 下載本專案的源碼。
3. 在專案的根目錄下運行 `dotnet restore` 來安裝依賴。
4. 運行 `dotnet run` 來啟動應用程式。

### 使用說明

1. 在瀏覽器中開啟 `http://localhost:5000`（或您設定的其他埠）。
2. 使用管理員帳號登入（預設帳號：admin@gmail.com，預設密碼：1Qaz2Wsx）。
3. 在後台管理頁面，您可以進行商品、類別和訂單的管理。
