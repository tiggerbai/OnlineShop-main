# 商業網站程式

這是一個典型的商業網站程式，前後端分離，使用 .NET 5.0 開發。

## 功能

- 管理員後台：預設一組管理者帳號，用於後台操作。管理者可以在後台上架商品。
- 商品展示：後台上架的商品會依據商品分類顯示於前端商場的輪播區塊。

## 安裝

1. 確保你的系統已經安裝了 .NET 5.0。
2. 下載本專案的源碼。
3. 在專案的根目錄下運行 `dotnet restore` 來安裝依賴。
4. 運行 `dotnet run` 來啟動應用程式。

訪問你的應用程式的 URL 來使用本專案。預設的管理者帳號為 `admin@gmail.com`，密碼為 `Masalu1993--`。
