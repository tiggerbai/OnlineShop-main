# OnlineShop 前台

這是 OnlineShop 商業網站的前台部分，使用 .NET 5.0 開發。

## 功能

- 商品展示：顯示後台上架的商品，商品會依據分類顯示於商場的輪播區塊。
- 商品評論與評分：用戶可以對商品進行評論和五星評分，並且商品會根據所收到的數據經過計算將獲得的平均分  數顯示於商品系資，訊提供更豐富的用戶互動和購物體驗。
- 購物車：用戶可以將商品加入購物車，並在購物車頁面進行結帳操作。
- 於結帳畫面輸入優惠代碼，系統會自動計算折扣金額。
- PayPal 付款整合：支援 PayPal 付款，並實現即時的付款資訊更新，提供安全便捷的線上支付體驗。
- 後台訂單管理：後台人員可以對已完成付款的訂單進行發貨操作，並產生提貨單據，實現高效的訂單處理流程。
- 重大的修改作業執行時皆會出現彈出訊息(Toast) 通知，提供更好的使用者體驗。

## 安裝

1. 確保你的系統已安裝 .NET Core 3.1 或以上版本以及 SQL Server。
2. 下載本專案的源碼。
3. 在專案的根目錄下運行 `dotnet restore` 來安裝依賴。
4. 運行 `dotnet run` 來啟動應用程式。

### 使用說明

1. 在瀏覽器中開啟 `http://localhost:5000`（或您設定的其他埠）。
2. 使用管理員帳號登入（預設帳號：admin@gmail.com，預設密碼：1Qaz2Wsx）。
3. 在後台管理頁面，您可以進行商品、類別和訂單的管理。
