# OnlineShop 前台

這是 OnlineShop 商業網站的前台部分，使用 .NET 5.0 開發。

## 功能

- 商品展示：顯示後台上架的商品，商品會依據分類顯示於商場的輪播區塊。

## 安裝

1. 確保你的系統已經安裝了 .NET 5.0。
2. 下載本專案的源碼。
3. 在專案的根目錄下運行 `dotnet restore` 來安裝依賴。
4. 運行 `dotnet run` 來啟動應用程式。

## 使用

訪問你的應用程式的 URL 來瀏覽商品和進行購物。
